The following pages detail a growing number of common experiments that can be done using the Rib:bit.

# [The BART Test](/experiments/bart-test/)
> The Balloon Analogue Risk Task, often shortened to BART, is a simple game-like activity about judging when to keep going and when to stop.

# [The Corsi Test](/experiments/corsi-test/)
> The Corsi Test is a simple memory game that asks someone to watch a pattern, hold it in their mind for a moment, and then repeat it back in the same order.
